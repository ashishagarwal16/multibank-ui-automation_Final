# MultiBank UI Automation Framework

## Overview
This repository contains a **production-ready Selenium UI automation framework** for testing critical user flows on the [MultiBank trading platform](https://trade.multibank.io/).
The framework is designed following **Page Object Model (POM)** principles, is **data-driven**, and supports **cross-browser execution**.

---

## **Tech Stack**

| Component | Technology |
|-----------|------------|
| Automation Tool | Selenium WebDriver |
| Programming Language | Java 11+ |
| Test Framework | TestNG |
| Build Automation | Maven |
| Design Pattern | Page Object Model (POM) |
| Reporting | TestNG Reports, ExtentReports (optional) |
| Browsers Supported | Chrome, Firefox |
